
# Frontend Boilerplate

A boilerplate of things that shouldn't exist.

## Contains

- [x] Webpack
- [x] React
- [x] Redux
- [x] Babel
- [x] Autoprefixer
- [x] CSS modules
- [x] Hot reloading
- [ ] Redux router
- [ ] Redux effects

## Setup

```
$ npm install
```

## Running

```
$ npm run start
```
