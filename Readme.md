
# Frontend Boilerplate

A boilerplate of things that mostly shouldn't exist.

## Contains

- [x] Webpack
- [x] React
- [x] Redux
- [x] Babel 6.x
- [x] Autoprefixer
- [x] CSS modules
- [x] Hot reloading
- [ ] Redux router
- [ ] Redux effects
- [x] TodoMVC example

## Setup

```
$ npm install
```

## Running

```
$ npm start
```
